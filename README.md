# دعوة زفاف أشرف و آمنة | Wedding Invitation

دعوة زفاف تفاعلية بالعربية مع تخطيط RTL وخطوط عربية فاخرة.

Interactive Arabic wedding invitation with RTL layout, luxury Arabic fonts, animations, and a celebration particle fountain.

**🔗 Repo**: `wedding-ghassen-2026`

## 🎉 المميزات | Features

- **شاشة افتتاح** بأنيميشن + موسيقى تبدأ عند الضغط
- **دعوة كاملة بالعربية** — بسم الله، الآية الكريمة، أسماء العائلتين، العريس والعروس
- **عد تنازلي حيّ** للأيام/الساعات/الدقائق/الثواني
- **خريطة Google ستالايت** لمكان الحفل
- **أزرار اتصال + واتساب** مباشرة
- **ختام بنافورة فراشات وبتلات** احتفالية
- **تجاوب كامل** مع كل أحجام الشاشات

## 🛠️ التقنيات | Tech Stack

- **Next.js 16** + TypeScript
- **Tailwind CSS 4** + Framer Motion
- **خطوط عربية**: Amiri, Aref Ruqaa, Reem Kufi, Cairo

## 🚀 التشغيل محلياً | Run Locally

```bash
bun install
bun run dev
```

ثم افتح http://localhost:3000

## 📦 النشر | Deploy

النشر على Vercel:

```bash
vercel
```

أو ارفع المشروع إلى GitHub ثم اربطه بـ Vercel.

## 📝 التخصيص | Customize

كل بيانات الزفاف موجودة في ملف واحد:
`src/lib/wedding-data.ts`

يمكنك تعديل:
- الأسماء (أشرف، آمنة، خالد، العائلة)
- التاريخ والوقت
- مكان الحفل وعنوانه
- أرقام التواصل
- النصوص (الآية، الدعاء، رسائل الختام)
- الموسيقى (استبدال ملف MP3)
